# Getting Started

This repo is built for two audiences:
- developers who want to understand how real agents are designed
- AI coding agents that need a reusable spec before implementation

## Fastest Path For Humans
1. Read [`what-is-an-ai-agent.md`](what-is-an-ai-agent.md).
2. Read [`agent-architecture-overview.md`](agent-architecture-overview.md).
3. Read [`production-agent-architecture-guide.md`](production-agent-architecture-guide.md).
4. Read one worked example under [`../examples/`](../examples/).
5. Use the skill to design your own agent.

## Fastest Path For AI Agents
1. Read [`../skills/production-agent-architecture/SKILL.md`](../skills/production-agent-architecture/SKILL.md).
2. Read [`../skills/production-agent-architecture/reference.md`](../skills/production-agent-architecture/reference.md).
3. Read [`../examples/marketing-agent/agent-build-spec.md`](../examples/marketing-agent/agent-build-spec.md).
4. Produce the required output artifacts.
