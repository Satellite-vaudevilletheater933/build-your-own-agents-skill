# What Is An AI Agent?

A chatbot answers once.

An agent works toward a goal over multiple steps.

## The Simple Mental Model
An agent is usually:
- a model
- plus tools
- plus session state
- plus a loop that decides what to do next
- plus permission and safety rules

## Why This Matters
If you only make one API call and print the answer, you have an LLM feature.

If the system can inspect state, use tools, take actions, evaluate results, and continue until a stop condition, you are much closer to an agent.
