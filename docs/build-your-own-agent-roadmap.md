# Build Your Own Agent Roadmap

## Stage 1: Define The Job
Write one sentence for:
- who the user is
- what the agent must do
- what counts as success

## Stage 2: Design The Loop
Define:
- when the model is called
- when tools are used
- what goes into state
- when the loop stops

## Stage 3: Design The Contracts
Create:
- tool contracts
- state shape
- permission matrix
- approval rules

## Stage 4: Add Production Basics
Before calling it production-ready, define:
- observability
- evaluation
- reliability and retries
- audit logging
