# Contributing

Contributions should make this repo more useful for developers or AI agents building production-ready systems.

## Good Contributions
- clearer docs
- better example agents
- stronger skill outputs
- more realistic production-readiness guidance
- cleaner install/use instructions

## Contribution Rules
1. Keep changes small and reviewable.
2. Prefer concrete examples over vague claims.
3. Update docs and skills together when the expected output changes.
4. Keep the repo easy for both humans and AI agents to parse quickly.
